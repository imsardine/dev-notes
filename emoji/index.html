Unicode ?? #

  • 第一次在 mkdocs.yml 用 site_name: "在電梯裡遇見雙胞胎 🚀" 會噴錯 unacceptable character #x1f680: special characters are not allowed in "/workspace/mkdocs.yml，但改用 escape sequence site_name: "在電梯裡遇見雙胞胎 \U0001F680" 就可以了。
  • Crashes on encountering emojis · Issue #48 · adrienverge/yamllint
    • adrienverge: (owner) 跟 pyyaml does not support literals in unicode over codepoint 0xffff · Issue #25 · yaml/pyyaml 有關
    • jayvdb: (contributor) 這問題在 PyYAML 已經被 merge 了，但一直沒 release，要不要改用 ruamel.yaml?
  • Support for UTF-16 surrogate pair encoded emojis · Issue #279 · go-yaml/yaml Go 也有類似的問題? #ril
  • python - Loading special characters with PyYaml - Stack Overflow Anthony Sottile: 似乎是 PyYAML 的 bug，改用 escape sequence 即可 #ril

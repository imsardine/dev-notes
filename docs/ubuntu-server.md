## 安裝設置 {: #setup }

### VirtualBox

  - 下載 Ubuntu Server LTS 的 ISO 檔。
  - 安裝時語言選英文、時區選 Other > Asia > Taiwan、而 locale 選 United States (`en_US.UTF-8`)
  - 磁碟分割採 Guided - use entire disk and set up LVM、採 No automatic updates、不要勾選 standard system utilities。
